# バージョン履歴（公開版）

SEALIB は2009年のデータ整理開始以来、木谷公哉（京都大学CSEAS）が継続開発
してきました。以下は公開可能な範囲で要約したバージョン履歴です。詳細な
実装内容や内部の管理手順は含みません。

| バージョン | 時期 | 主な内容 |
|-----------|------|----------|
| v0.x | 2010年 | データインポート・検索・多言語UIの基礎構築 |
| v1.0〜v1.11 | 2011〜2012年 | プロトタイプ公開（雑誌のみ） |
| v2.0 | 2012〜2014年 | 新聞・官報を追加、書誌ID体系を整備、公開版リリース |
| v2.1〜v2.4 | 2015〜2020年 | ベトナム語・ラオ語・クメール語・ビルマ語UIを追加、増補版（Extended Mode）を導入、ベトナム・ラオス・カンボジアのデータを追加 |
| v3.0〜v3.1 | 2021〜2025年 | PHP 8対応、少数民族言語データの追加、データ品質の継続的改善 |
| v3.2.x | 2026年4月 | Journal Metrics（外部評価指標）機能を追加。SINTA等の指標を書誌情報に紐づけ、検索・詳細表示に対応 |
| v3.3.0 | 2026年4月 | REST API v1 と OAI-PMH 2.0 を公開。多言語の利用マニュアルを整備 |
| v3.3.1 | 2026年5月 | REST API v1 に Journal Metrics 連携（`include=metrics`、`metric_source` 絞り込み）を追加 |
| **v3.4.0** | **2026年7月** | Journal Metrics のインポート処理をパイプラインとして正式化。SINTA（インドネシア）データを追加取り込み |

## データ収録の広がり

- 2011年: 日本のコアジャーナル（雑誌）データから開始
- 2014年: 新聞・官報を追加
- 2015〜2019年: ベトナム・ラオス・カンボジアの協力機関データを順次追加
- 2020年代: 協力研究者による大陸部少数民族言語データの追加、外部機関との
  データ連携（NII等）
- 現在: 収録機関120、書誌13,752件（増補版）、所蔵17,259件
