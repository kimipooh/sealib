# 東南アジア逐次刊行物総合目録データベース（SEALIB）の概要

## 目的

東南アジア逐次刊行物総合目録データベース（SEALIB）は、東南アジア諸国および
日本の図書館・研究機関が所蔵する逐次刊行物（雑誌・新聞・官報）の書誌情報と
所蔵情報を統合的に検索・公開するためのデータベースです。2011年のプロトタイプ
公開以来、120の協力機関とともに構築されてきました。

SEALIB という名称は、当初の Southeast Asian Library Database という構想に
由来します。現在の公開文書では、東南アジア逐次刊行物総合目録データベースを
指す略称として用いています。

## 地域研究における位置づけ

東南アジア地域の逐次刊行物は、書誌形式の不統一や特殊文字（少数言語の
フォント等）の扱いなど、既存の大規模書誌データベースの仕様には馴染みに
くい特性を持っています。SEALIB は、こうしたデータの実情に合わせてシステム
側を調整するアプローチを採り、東南アジア地域研究における逐次刊行物情報の
統合的な参照基盤として運用されています。

## 提供する情報

SEALIB は、以下の情報を横断的に参照できる情報基盤です。

- **書誌情報**: タイトル、責任表示、刊行頻度、出版国・言語など
- **所蔵情報**: どの機関がどの巻号を所蔵しているか
- **外部評価指標（Journal Metrics）**: SINTA など外部の学術誌評価情報を
  書誌情報に紐づけて参照可能（詳細は
  [journal-metrics.md](journal-metrics.md) を参照）

## 提供形態

SEALIB は、Web画面での利用と、API による機械的な利用の両方に対応して
います。

### 画面での利用

多言語対応のWeb検索インターフェースから、次のことができます。

- 逐次刊行物を検索する（キーワードをスペース区切りまたは `AND` でつなぐ
  AND検索、`OR` を使った OR検索に対応。大文字・小文字は区別せず、部分
  一致で検索されます）
- 音標記号のあり・なし、どちらの表記でも検索する（例: 音標付きの綴りで
  も、音標を外した綴りでも同じレコードを検索できます）
- 書誌情報を確認する
- 所蔵情報（どの機関がどの巻号を所蔵しているか）を確認する
- 詳細ページで Journal Metrics（外部評価指標）を確認する
- 詳細ページから CiNii Books・国立国会図書館サーチ・Library of Congress
  など外部サービスへ移動する

検索方法・検索結果画面・詳細画面の詳しい説明は
[user-guide.md](user-guide.md) を参照してください。

### API・データ連携としての利用

REST API v1（読み取り専用、JSON）と OAI-PMH 2.0（メタデータハーベス
ティング）を通じて、次のことができます。

- 書誌情報・所蔵情報を機械的に取得する
- `include=metrics` を指定して Journal Metrics を付与した形で取得する
- `metric_source` を指定して特定の評価情報源に絞り込む
- 取得したデータを分析や外部システムとの連携に利用する

API の詳細は [api-overview.md](api-overview.md) を参照してください。
