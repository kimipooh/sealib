# 利用ガイド（画面での使い方）

本ページは、東南アジア逐次刊行物総合目録データベース（SEALIB）のWeb
検索画面の使い方をまとめたものです。実装の詳細やサーバー内部の構成には
触れません。API・OAI-PMH での機械的な取得方法は
[api-overview.md](api-overview.md) を参照してください。

## 検索方法

- **AND検索**: 複数のキーワードをスペースで区切る、または `AND` で
  つなぐと、すべてのキーワードを含むレコードに絞り込めます。
- **OR検索**: `OR` でキーワードをつなぐと、いずれかのキーワードを含む
  レコードを検索できます。
- **大文字・小文字を区別しない**: アルファベットの大文字・小文字は区別
  されません。
- **部分一致**: 入力した文字列を含むレコードが検索されます（完全一致は
  必要ありません）。
- **音標記号あり・なしの両方に対応**: ベトナム語やその他の言語で使われる
  音標記号（アクセント記号など）について、音標を含む綴りと、音標を外した
  綴りのどちらで入力しても、同じレコードを検索できます。

## 検索項目

主な検索項目は次のとおりです。

- キーワード（タイトル・著者などを横断的に検索）
- 所蔵館
- 国・地域
- メディア種別（雑誌・新聞・官報など）
- ISSN

これ以外にも、公開画面にはいくつかの検索項目が用意されています。詳細は
実際の検索画面でご確認ください。

## 検索結果画面

検索結果一覧には、次の情報が表示されます。

- **No.**: 結果内の通し番号
- **タイトル事項**: タイトルおよび責任表示
- **リンクの有無**: 詳細情報や外部リンクの有無
- **表示件数**: 1ページあたりの表示件数
- **タイトル順ソート**: タイトルのアルファベット順での並べ替え

## 詳細画面

検索結果からレコードを選択すると、詳細画面で次の情報を確認できます。

- **書誌情報**: タイトル、責任表示、刊行頻度、出版国・言語など
- **所蔵館情報**: どの機関がどの巻号を所蔵しているか
- **外部サービスへのリンク**: NCID、ISSN、LCCN、NDL など、関連する識別子
  ・外部サービスへのリンク
- **Journal Metrics**: 対応する外部評価指標がある場合は、評価元・
  グレードなどの情報もあわせて表示されます（詳細は
  [journal-metrics.md](journal-metrics.md) を参照）

## 収録データ

SEALIB には、収録範囲の異なる2つの版があります。

- **コアジャーナル版**: 中核となる雑誌を中心とした、絞り込んだ収録範囲
- **増補版**: コアジャーナル版に加え、新聞・官報や協力機関から提供された
  データを含む、より広い収録範囲

所蔵情報は、データ整備・調査を行った時点の情報です。最新の所蔵状況を
確認する場合は、各所蔵館のOPAC等をあわせてご確認ください。

## 音標処理

本データベースは、音標記号を含む表記について、音標あり・なしのどちらの
入力でも検索できるように対応しています。対応関係を示す音標変換表も、
Web画面上で公開しています。内部でどのように処理しているかの詳細は、
本ドキュメントでは扱いません。

## 外部サービス連携

詳細画面からは、次のような外部サービスへ移動できます。

- CiNii Books
- タイトル変遷マップ表示システム
- 国立国会図書館（NDL）サーチ
- Library of Congress
- 所蔵機関によっては、各機関のOPAC（例: VASS OPAC）

## API・OAI-PMH での利用

画面での検索・閲覧に加えて、SEALIB は REST API v1 と OAI-PMH 2.0 による
機械的なデータ取得にも対応しています。書誌情報・所蔵情報に加え、
`include=metrics` の指定により Journal Metrics も取得できます。詳細は
[api-overview.md](api-overview.md) を参照してください。
