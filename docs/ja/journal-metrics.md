# Journal Metrics（外部評価指標連携）

## 目的

SEALIB は、書誌情報・所蔵情報に加えて、学術誌の外部評価指標（Journal
Metrics）を参照できるようにしています。これにより、利用者は特定の逐次
刊行物について「どこで所蔵されているか」だけでなく「どのように評価されて
いるか」も合わせて確認できます。

## 現在の連携状況

現在対応している外部評価情報源の一例として、インドネシアの学術誌評価制度
である SINTA があります。SINTA の指標データを、書誌情報に半自動的に取り
込む仕組みを整えています。取り込まれた評価指標は、Web検索結果の詳細ページ
および REST API（`include=metrics`）から参照できます。

## 確認方法

Journal Metrics に取り込まれた評価指標は、次の2通りの方法で参照できます。

- **画面での確認**: 書誌詳細ページを開くと、対応する評価指標（評価元、
  グレード、参照URLなど）が書誌情報・所蔵情報とあわせて表示されます。
- **API での取得**: REST API v1 で `include=metrics` を指定すると、
  書誌情報を取得する際に評価指標を付与した形で取得できます。特定の情報源
  （例: SINTA）に絞り込みたい場合は `metric_source` を指定します。詳細は
  [api-overview.md](api-overview.md) を参照してください。

## 汎用的な枠組みとしての設計

Journal Metrics は、特定の国や評価制度に限定されない汎用的な枠組みとして
設計しています。SINTA のような学術誌評価指標に加え、各国・各機関が提供する
関連情報を、書誌情報・所蔵情報とあわせて参照できるようにすることを目的と
しています。評価元ごとに指標の種類や粒度が異なっていても、`metric_source`
という共通の切り口で識別・絞り込みできる枠組みになっているため、新しい
評価情報源を追加する際にも、既存の書誌情報・所蔵情報の構造や検索方法を
変更する必要はありません。
