# Version History (Public)

SEALIB has been developed continuously by Kimiya Kitani (CSEAS, Kyoto
University) since data-organization work began in 2009. The table below
is a public-safe summary of the version history. It does not describe
implementation details or internal maintenance procedures.

| Version | Period | Summary |
|---------|--------|---------|
| v0.x | 2010 | Initial import pipeline, search, and multilingual UI foundation |
| v1.0–v1.11 | 2011–2012 | Public prototype release (journals only) |
| v2.0 | 2012–2014 | Added newspapers and official gazettes, formalized the bibliographic ID scheme, public release |
| v2.1–v2.4 | 2015–2020 | Added Vietnamese, Lao, Khmer, and Burmese UI; introduced the Extended (comprehensive) edition; added data from partner institutions in Vietnam, Laos, and Cambodia |
| v3.0–v3.1 | 2021–2025 | PHP 8 compatibility; added minority-language serials data; ongoing data-quality improvements |
| v3.2.x | April 2026 | Added Journal Metrics: external evaluation indices (e.g. SINTA) linked to bibliographic records, with search and detail-page support |
| v3.3.0 | April 2026 | Released REST API v1 and OAI-PMH 2.0; published bilingual usage documentation |
| v3.3.1 | May 2026 | Added Journal Metrics support to REST API v1 (`include=metrics`, `metric_source` filtering) |
| **v3.4.0** | **July 2026** | Formalized the Journal Metrics import process as a repeatable pipeline; added further SINTA (Indonesia) data |

## Growth of coverage

- 2011: Started with Japan-held core journal data
- 2014: Added newspapers and official gazettes
- 2015–2019: Progressively added data from partner institutions in
  Vietnam, Laos, and Cambodia
- 2020s: Added minority-language serials data through collaborating
  researchers; integrated data from external institutions (e.g. NII)
- Current: 120 partner institutions, 13,752 bibliographic records
  (Extended edition), 17,259 holdings records
